So what's the deal?

  • Javascript = prototye

What is Inheritance?

  • dictionary: "something that is or may be inherited"
  • One obejct get access to the properties and methods of another object.

Prototye

  • Everything except primitives are objects in Javascript
  • Most, if not all, objects are instances of `Object`
  • Inherits from objects, ie. `Object.create()`
  • prototype: template object which is linked to another object
  • "differential inheritance": creates invisible link to the parent object

Classical

  • classes can only extend from other classes
  • classes are immutable at runtime
  • base classes and derived classes: dervied inherits base classes
  • traditionally, the derived class will create overrides
  • This is truely not in javascript, but can be imitated via prototypal
